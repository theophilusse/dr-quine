# dr-quine

> *This project is about making you discover the recursion theorem of Kleene.*

A **quine** is a program that produces its own source code as output, without reading any file or taking any input. This project implements three quine programs — **Colleen**, **Grace**, and **Sully** — each with different properties, in **C**, **x86-64 Assembly**, and **Python** (bonus).

---

## Programs

### Colleen
Prints its own source code to stdout.
```bash
# C
./c/Colleen > tmp && diff tmp Colleen.c
# ASM
./asm/Colleen > tmp && diff tmp Colleen.s
# Python
python3 ./python/Colleen.py > ./python/tmp && diff ./python/tmp ./python/Colleen.py
```

### Grace
Writes its own source code into a file named `Grace_kid.c` / `Grace_kid.s` / `Grace_kid.py`.
```bash
# C
./c/Grace && diff Grace.c Grace_kid.c
# ASM
./asm/Grace && diff Grace.s Grace_kid.s
# Python
python3 ./python/Grace.py && diff ./python/Grace.py ./pythton/Grace_kid.py
```

### Sully
Writes a modified copy of itself, compiles/runs it, and decrements the integer `X` starting at `5`. Execution stops when `X < 0`.
```bash
# C
./c/Sully
ls Sully_*.c   # Sully_5.c Sully_4.c ... Sully_0.c
# ASM
./asm/Sully
ls Sully_*.s   # Sully_5.s Sully_4.s ... Sully_0.s
# Python
python3 ./python/Sully.py
ls ./python/Sully_*.py  # Sully_5.py Sully_4.py ... Sully_0.py
```

## Structure

```
dr-quine/
├── Colleen.c / Colleen.s
├── Grace.c   / Grace.s
├── Sully.c   / Sully.s
├── Makefile
├── asm/                  # ASM executables (generated by make)
├── c/                    # C executables (generated by make)
├── python/               # Python scripts (bonus)
    ├── Colleen.py
    ├── Grace.py
    └── Sully.py
└── tool/
    ├── Colleen.asm.c     # C source used to generate Colleen.s
    ├── Grace.asm.c       # C source used to generate Grace.s
    ├── Sully.asm.c       # C source used to generate Sully.s
    ├── generate.c        # Tool: encodes a C source into a code[] string
    ├── generate_asm.c    # Tool: encodes an ASM source into .ascii chunks
    └── assemble.sh       # Helper script to compile ASM sources
```

## Usage

```bash
# Build everything
make

# Run C versions
./c/Colleen
./c/Grace
./c/Sully

# Run ASM versions
./asm/Colleen
./asm/Grace
./asm/Sully

# Clean object files
make clean

# Full clean
make fclean
```

---

## Build

| Rule | Description |
|------|-------------|
| `make` / `make all` | Compiles all C and ASM programs |
| `make clean` | Removes object files |
| `make fclean` | Removes object files and binaries |
| `make re` | `fclean` + `all` |

**Requirements:**
- `gcc` with `-Wall -Wextra -Werror`
- `as` (GNU Assembler) — x86-64 Linux

---

## Key Concepts

- **Quine** — a self-reproducing program that uses no file I/O or external input
- **Kleene's recursion theorem** — the theoretical foundation proving that self-referential programs can exist
- **Fixed point** — the core trick: a data structure that encodes itself, used as the program's own description
