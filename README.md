# dr-quine

> *This project is about making you discover the recursion theorem of Kleene.*

A **quine** is a program that produces its own source code as output, without reading any file or taking any input. This project implements three quine programs — **Colleen**, **Grace**, and **Sully** — each with different properties, in both **C** and **x86-64 Assembly**.

---

## Programs

### Colleen
Prints its own source code to stdout.
```
./Colleen > tmp && diff tmp Colleen.c
./Colleen > tmp && diff tmp Colleen.s
```

### Grace
Writes its own source code into a file named `Grace_kid.c` / `Grace_kid.s`.
```
./Grace && diff Grace.c Grace_kid.c
./Grace && diff Grace.s Grace_kid.s
```

### Sully
Writes a modified copy of itself into `Sully_X.c` / `Sully_X.s`, compiles it and runs it. The integer `X` starts at `5` and decrements at each generation. Execution stops when `X < 0`.
```
./Sully
ls Sully_*.c   # Sully_5.c Sully_4.c ... Sully_0.c
```

## Structure

```
dr-quine/
├── Colleen.c / Colleen.s
├── Grace.c   / Grace.s
├── Sully.c   / Sully.s
├── Makefile
├── asm/                  # ASM executables (generated by make)
├── c/                    # C executables (generated by make)
└── tool/
    ├── Colleen.asm.c     # C source used to generate Colleen.s
    ├── Grace.asm.c       # C source used to generate Grace.s
    ├── Sully.asm.c       # C source used to generate Sully.s
    ├── generate.c        # Tool: encodes a C source into a code[] string
    ├── generate_asm.c    # Tool: encodes an ASM source into .ascii chunks
    └── assemble.sh       # Helper script to compile ASM sources
```

## Usage

```bash
# Build everything
make

# Run C versions
./c/Colleen
./c/Grace
./c/Sully

# Run ASM versions
./asm/Colleen
./asm/Grace
./asm/Sully

# Clean object files
make clean

# Full clean
make fclean
```

---

## Build

| Rule | Description |
|------|-------------|
| `make` / `make all` | Compiles all C and ASM programs |
| `make clean` | Removes object files |
| `make fclean` | Removes object files and binaries |
| `make re` | `fclean` + `all` |

**Requirements:**
- `gcc` with `-Wall -Wextra -Werror`
- `as` (GNU Assembler) — x86-64 Linux

---

## Key Concepts

- **Quine** — a self-reproducing program that uses no file I/O or external input
- **Kleene's recursion theorem** — the theoretical foundation proving that self-referential programs can exist
- **Fixed point** — the core trick: a data structure that encodes itself, used as the program's own description
